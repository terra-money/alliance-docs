This testnet use the version of [Alliance v0.0.1-beta1](https://github.com/terra-money/alliance/releases/tag/v0.0.1-beta1) and [CosmosSDK 0.46.3](https://github.com/cosmos/cosmos-sdk/releases/tag/v0.46.3). 

## Download or compile Alliance

To join the testnet you must download the binary or compile the source code from [v0.0.1-beta1 tag from Alliance repository](https://github.com/terra-money/alliance/releases/tag/v0.0.1-beta1).

<Admonition type="tip" title="To compile the code">
    Install [Ignite Cli v0.25.1](https://docs.ignite.com/guide/install) and use the command <b>ignite chain build</b>.
</Admonition>

## Join the testnet

Joining the testnet follows the standard process from CosmosSDK and Terra has a very well defined guide on how to [join a testnet here](https://docs.terra.money/full-node/manage-a-terra-validator/). Take in consideration that if you follow terra guide you must:

- use **allianced** instead of **terrad** command,
- use the following peers [from here](http://3.75.187.158:26657/net_info),
- use the following [genesis state](http://3.75.187.158:26657/genesis?).
- [faucet to request funds](http://3.75.187.158) to be able to submit transactions and create the validator 

<Admonition type="tip" title="Request tokens">
    In case you need more tokens please go to [Terra Validators Discord and requesting for tokens for Alliance Testnet](https://discord.gg/Bj4NpaqKWb).
</Admonition>
