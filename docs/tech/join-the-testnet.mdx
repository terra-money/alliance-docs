This testnet consists of four chains `ordos`, `corrino`, `harkonnen` and `atreides`.

## 1. Download or compile the chain binary

To join the testnet, download the binary or compile the source code from the [v0.0.1-goa tag of the Alliance repository](https://github.com/terra-money/alliance/releases/tag/v0.0.1-goa).

<Admonition type="tip" title="Install ignite">

To compile the code for a particular chain, replace the ACC_PREFIX with one of the four chains above.

```sh bash
git clone https://github.com/terra-money/alliance.git && cd alliance
git checkout v0.0.1-goa
make build-alliance ACC_PREFIX=ordos
```

The binaries will be found in the `build` folder.

</Admonition>

## 2. Join the testnet

Joining the Game of Alliance testnet follows the same process as other CosmosSDK testnets. Terra has a very well-defined guide on how to [join a testnet](https://docs.terra.money/full-node/manage-a-terra-validator/).

If you use the [Terra testnet guide](https://docs.terra.money/full-node/manage-a-terra-validator/) you must:

- Use _`ordosd | corrinod | atreides | harkonnen`_ instead of _`terrad`_ commands.
- Use the genesis file for the chain you want to join: [https://github.com/terra-money/alliance/tree/feat/wasm-testnets/genesis](https://github.com/terra-money/alliance/tree/feat/wasm-testnets/genesis) .
