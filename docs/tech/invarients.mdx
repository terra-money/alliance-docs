# Invariants

## Validator Shares

This invariant checks that the sum of validator shares of an asset owned by all validators is equal to the total shares recorded in the corresponding Alliance asset. It also checks for negative shares. For more on validator shares, visit the [Concepts section](../concepts/delegation.mdx)
